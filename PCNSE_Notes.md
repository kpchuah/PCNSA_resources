## IP Routing
The Administrative Distances assosiated with routing are:</br>
Routing Type --- Administrative Distance</br>
Connected ------- 0</br>
Static ---------- 10</br>
e BGP ----------- 20</br>
i OSPF ---------- 30</br>
e OSPF ---------- 110</br>
RIP ------------- 120</br>
i BGP ----------- 200</br>
</br>
In Palo Alto CLI, you can find this by the command `show routing summary`</br>
